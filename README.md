# SHEBot

A helpful little Slack bot for SHE Choir to take attendance and remind us about rehearsals.

## Installation

Slack permissions needed:

- bot
- commands
- incoming-webhook
- chat:write:bot
- reactions:read
- reactions:write
- emoji:read
- users:read
